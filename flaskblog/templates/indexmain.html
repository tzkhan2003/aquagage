what we do

Our main purpose is to provide a tool to value & conserve water. Tracking and saving water using IoT for every user. Monitor & highlight water usage behavior. Suggest to train behaviour using Machine Learning. Potentially reuse greywater.

why us

We intended to install the system in every water access points. This system can be used in existing infrastructure. But the system can work efficiently if implemented while planning. The system is optimum for urban households.
